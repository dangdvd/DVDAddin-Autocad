# DVDAddin-Autocad
Tiện ích mở rộng cho autocad

Tiện ích có các lệnh sau:

DVDQD1 : Tự động rải 1 lớp Dim chi tiết qua các điểm giao cắt.

DVDQD2 : Tự động rải 2 lớp Dim (Chi tiết + Tổng).

DVDQD3 : Tự động rải 3 lớp Dim (Chi tiết + Trục + Tổng).

DVDQDP : Tự động rải Dim đo chiều dài các đoạn của Polyline hoặc Line.

DVDC2E : Xuất tọa độ Text/Point/Block ra file CSV.

DVDC2T : Xuất tọa độ Text/Point/Block ra file TXT (tab-delimited).

DVDE2C : Nhập tọa độ từ file CSV → vẽ Text + Point.

DVDT2C : Nhập tọa độ từ file TXT (tab-delimited) → vẽ Text + Point.

DVDPRINT : In hàng loạt nhiều Layout/Model ra PDF hoặc máy in.

DVDTRA : Dịch Text/MText/Attribute sang ngôn ngữ khác.

DVDCF : Chuyển đổi bảng mã (Unicode, TCVN3, VNI) cho văn bản.

DVDSETGEMINI: Cài đặt API key và model cho tính năng dịch bằng Gemini.


Tác giả: Đặng Văn Đằng
